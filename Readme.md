# vidTube
vidTube is a video streaming website made using nodeJS and mongoDb

 <img src="demo_vidTube.png" width="100%" alt="demo_vidTube">

# Features
 • Like, dislike, share, download, report a video
 
 • Create account and upload videos

 • Admin panel for managing all users

 • Managing reported videos in admin panel i.e either delete video or discard report

# Accessing admin panel
There is same login system for both users and admins , to become a admin set user role in database (mongoDb) to admin i.e ```role:"admin"```
